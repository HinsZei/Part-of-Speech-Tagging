How to run the project:

IDE: enter the IDE and run the main.py

Command Line: enter the root directory of the project, then type the following command:
		python main.py <en|es|nl|cantonese>
	         prarams are optional, the default param is 'en'

How to select another treebank:

IDE: edit configuration -> parameters -> en|es|nl|cantonese or edit variable 'lang' in the main.py

Command Line: mentioned above.

The project can't handle the wrong inputs, so please don't try other params XD.
